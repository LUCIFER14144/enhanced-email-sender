- JWT token authentication
- bcrypt password hashing
- Input validation
- SQL injection & XSS prevention

## System Requirements

- 4GB RAM minimum
- Internet connection
- 200MB free disk space

- Enhanced-Email-Sender.exe (82 MB) - Main application
- Enhanced-Email-Sender.zip (smaller) - Complete package with setup scripts
