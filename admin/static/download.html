Getting Started

  1. Download the ZIP using the button above
  2. Extract all files to a folder (e.g., C:\EmailSender\)
  3. Run SETUP_WKHTMLTOPDF.bat as administrator (for PDF features)
  4. Double-click Enhanced-Email-Sender.exe
  5. Login with credentials provided by admin
  6. Start sending emails with cloud sync!
Note: wkhtmltopdf setup is optional but recommended for best PDF/Image quality. The app will work without it using fallback methods.

If you encounter any issues:

  • Windows may flag new executables from the internet. Prefer the ZIP download above, then extract and run the EXE.
  • If SmartScreen appears, click "More info" → "Run anyway" (only if you trust this app).
  • Make sure you have your login credentials from the admin
  • Verify your internet connection for cloud features
  • Contact your administrator if login issues persist
