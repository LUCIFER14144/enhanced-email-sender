# Enhanced Email Sender - Package Contents

This package includes:

## Files:
- **Enhanced-Email-Sender.exe** - Main application (82 MB)
- **SETUP_INSTRUCTIONS.txt** - Detailed setup guide
- **INSTALL_WKHTMLTOPDF.bat** - Automatic installer for wkhtmltopdf
- **README.md** - This file

## wkhtmltopdf folder:
- Contains wkhtmltopdf installer (download separately or use provided)
- Required for PDF and image generation features
- Installation takes about 30 seconds
